### プログラム概要

トレード練習プログラムです。
直感的な操作で練習ができます。

操作方法
- 左右ボタン：バーの移動
- 上下ボタン：新規買い/売り
- Enterキー：決済

注意点
- 両建はできません。成行注文のみ可能です。
- クロス円またはドルストレートのみ対応しています。
